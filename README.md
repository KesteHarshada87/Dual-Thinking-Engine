# 🧠 Human vs 🤖 Machine Thinking Visualizer

A creative JavaScript project that compares **how humans think emotionally vs how machines think logically** — powered by AI.
---------------------------------------------------------------------------------------------------------------

## 🚀 Project Overview

This project allows users to enter any problem or situation and visualizes two different types of thinking:

* 👤 **Human Thinking** → Emotional, overthinking, confused
* 🤖 **Machine Thinking** → Logical, structured, decision-based
------------------------------------------------------------------------------------------------------------

## ✨ Features

* 🤖 AI-powered responses using Groq API
* 👤 Emotional human-like thinking simulation
* 🧠 Logical machine-based reasoning
* ⏱️ Step-by-step animated output
* 🎨 Clean and simple UI
* ⚡ Real-time input-based results
----------------------------------------------------------------------------------------------------------

## 🛠️ Tech Stack

## ⚙️ How It Works

1. User enters a problem
2. Input is sent to Groq API
3. Two prompts are generated:

   * Human-style thinking
   * Machine-style thinking
4. AI returns responses
5. Output is displayed step-by-step in UI
-------------------------------------------------------------------------------------------------------------
